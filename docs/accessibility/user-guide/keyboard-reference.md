## Custom Shortcuts

You can customize most shortcuts through the Accessibility preferences:

1. Press `Ctrl+Shift+,` to open keyboard customization
2. Navigate to the action you want to modify
3. Press `Enter` to record new shortcut
4. Press your desired key combination
5. Press `Enter` to confirm or `Escape` to cancel

## Shortcut Conflicts

If shortcuts conflict with system or ORCA shortcuts:

1. XFB shortcuts take priority within the XFB window
2. System shortcuts (like `Alt+Tab`) always work globally
3. ORCA shortcuts (like `Insert+H`) work when ORCA has focus
4. Use `Ctrl+Shift+,` to resolve conflicts by changing XFB shortcuts

## Tips for Efficient Use

1. **Learn the Alt+Letter shortcuts** for quick area navigation
2. **Use Ctrl+T frequently** during broadcasting to monitor time
3. **Customize shortcuts** for your most-used functions
4. **Practice grid navigation** with arrow keys for efficient library browsing
5. **Use search (Ctrl+F)** instead of manual browsing for large libraries
