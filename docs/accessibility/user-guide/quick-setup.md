# XFB Quick Setup Guide for ORCA Users

## 5-Minute Setup

This guide gets you up and running with XFB and ORCA in just a few minutes.

### Step 1: Verify ORCA is Running

1. Press `Insert+Space` - you should hear ORCA preferences dialog
2. If ORCA isn't running, press `Alt+F2`, type `orca`, press Enter
3. Close ORCA preferences with `Escape`

### Step 2: Launch XFB

1. Press `Alt+F2`, type `xfb`, press Enter
2. Wait for "XFB Radio Broadcasting Software loaded with accessibility support"
3. If you don't hear this message, see troubleshooting section

### Step 3: Test Basic Navigation

1. Press `Tab` - ORCA should announce the first control
2. Press `Tab` several more times to navigate through interface
3. Press `Alt+M` to access the menu bar
4. Press `Escape` to return to main interface

### Step 4: Add Music to Library

1. Press `Ctrl+Shift+L` to open library management
2. Press `Ctrl+A` to add music directory
3. Navigate to your music folder and press Enter
4. Wait for "Library scan completed" announcement

### Step 5: Create Your First Playlist

1. Press `Ctrl+N` to create new playlist
2. Type a name for your playlist and press Enter
3. Navigate to music library with `Alt+L`
4. Select tracks with arrow keys and `Ctrl+Shift+A` to add to playlist

### Step 6: Test Playback

1. Press `Alt+P` to go to playlist area
2. Use arrow keys to select a track
3. Press `Space` to start playback
4. Press `Ctrl+T` to hear current time
5. Press `Space` again to pause

## Essential Shortcuts to Remember

- `Space` - Play/Pause
- `Ctrl+T` - Announce current time
- `Ctrl+,` - Open preferences
- `F1` - Help
- `Alt+F4` - Exit
